# how to add image

## 保存先
- events: `public/images/events/`
- idols: `public/images/idols/`
- banners: `public/images/banners/`
- lab: `public/images/lab/`
- news: `public/images/news/`
- placeholders: `public/images/placeholders/`

## 推奨サイズ
- event: 1200x675
- idol: 800x800 or 1200x900
- banner: 1600x600
- lab: 1200x675

## ルール
- JSONには `/images/...` 形式で記載
- 日本語ファイル名は避ける
- jpg / webp 推奨
